# Terraria 服务器搭建
非常简单，只需要进入官网，找到服务器组件（网站下面），使用`wget`下载即可！
1. 进入[Terraria 官方网站](https://terraria.org/)
2. 找到网站下方的[PC Dedicated Server](https://terraria.org/api/download/pc-dedicated-server/terraria-server-1449.zip)
3. 使用`wget https://terraria.org/api/download/pc-dedicated-server/terraria-server-1449.zip`
4. 解压并将文件给予执行权限
5. Terraria, 启动！
